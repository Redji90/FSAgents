# Figure Skating Bot

Этот проект представляет собой телеграмм-бота, который анализирует результаты по фигурному катанию и предоставляет базовую аналитику за сезон. Бот также работает с PDF-документами и подключен к LLM для получения расширенной аналитики и инсайтов.

## Функциональность

- **Анализ результатов**: Бот обрабатывает результаты соревнований по фигурному катанию и предоставляет статистику.
- **Работа с PDF**: Извлечение и парсинг данных из PDF-документов с результатами.
- **Интерактивность**: Пользовательский интерфейс с настраиваемыми клавиатурами для удобного взаимодействия.
- **Подключение к LLM**: Использование модели для получения дополнительных аналитических данных.

## Установка

1. Клонируйте репозиторий:
   ```
   git clone <URL>
   ```
2. Перейдите в директорию проекта:
   ```
   cd figure-skating-bot
   ```
3. Установите зависимости:
   ```
   pip install -r requirements.txt
   ```

## Настройка

Создайте файл `.env` на основе `.env.example` и заполните необходимые переменные окружения.

## Запуск

Для запуска бота выполните:
```
python src/main.py
```

## Тестирование

Тесты находятся в директории `tests`. Для запуска тестов используйте:
```
pytest
```
